## Overview

This repository contains the complete research submission for a literature review-based research paper examining the cryptographic vulnerabilities of blockchain systems in the context of emerging quantum computing capabilities.

The central argument is that while **decentralization** protects blockchain networks against conventional attacks, it provides **no defense** against quantum-level cryptographic key extraction — making post-quantum migration both urgent and technically complex.

## Research Type

**Literature Review & Conceptual Analysis**  
All findings are synthesized from peer-reviewed publications, NIST standards, and technical reports. No primary experiments or original mathematical derivations are conducted.

---

## Key Topics Covered

- Elliptic Curve Digital Signature Algorithm (ECDSA) and its quantum vulnerability
- Shor's Algorithm — breaking public-key cryptography in polynomial time
- Grover's Algorithm — halving the security of cryptographic hash functions
- "Harvest Now, Decrypt Later" threat vector on immutable blockchain records
- Post-quantum cryptographic alternatives: **CRYSTALS-DILITHIUM**, **FALCON**, **SPHINCS+**, **CRYSTALS-Kyber**
- Migration feasibility: technical, governance, and computational challenges
- A 4-phase conceptual Quantum Threat Assessment Framework
